Ejercicio #5

En que departamento hay más usuarios registrados

  • departamento
  • Id
Ver código
SELECT departments.name, departments.id, COUNT(departments_id) as numero_de_usuarios FROM `users` INNER JOIN departments WHERE users.departments_id = departments.id GROUP BY departments_id ORDER BY numero_de_usuarios DESC LIMIT 1;

Ejercicio #8

  • Nombres
  • Apellidos
  • Correo

  • Nombre del departamento
  • Nombre del país
Ver código
SELECT users.names, users.lastname, users.email, schools.name as escuela, departments.name as departamento, countries.name as pais FROM `users` INNER JOIN schools INNER JOIN departments INNER JOIN countries WHERE users.school_id = schools.id and users.departments_id = departments.id and departments.countries_id = countries.id ORDER BY rand() LIMIT 1;
